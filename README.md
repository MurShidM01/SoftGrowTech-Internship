Modern, focused internship tasks and mini-projects built at SoftGrowTech. Each project is self-contained and includes a dedicated README with purpose, features, and run steps.

**Internship Tasks**

- Built small, practical utilities to strengthen Python fundamentals.
- Focused on clean I/O, regex usage, and interactive CLI behavior.
- Structured each task as a standalone project with clear documentation.

**Projects**

1. **Simple Chatbot**
   A basic command-line chatbot that responds to a small set of predefined inputs.
   Folder: `Simple-Chatbot`
   README: `Simple-Chatbot/README.md`

2. **Email Extractor**
   A Python script that extracts email addresses from a text file and writes them to an output file.
   Folder: `Email-Extractor`
   README: `Email-Extractor/README.md`

3. **Geolocation**
   A Python tool that fetches IP location data and generates an interactive map.
   Folder: `Geolocation`
   README: `Geolocation/README.md`

4. **Memory Puzzle**
   A GUI-based memory matching card game with timer and emoji cards.
   Folder: `Memory-Puzzle`
   README: `Memory-Puzzle/README.md`

5. **Expense Tracker**
   A CLI-based personal expense tracker to add, view, and calculate expenses.
   Folder: `Expense-Tracker`
   README: `Expense-Tracker/README.md`

**Repository Structure**

- `Simple-Chatbot/`
- `Email-Extractor/`
- `Geolocation/`
- `Memory-Puzzle/`
- `Expense-Tracker/`
- `README.md`

**Notes**

- Each project is independent and can be run on its own.
- Outputs are written to local files where applicable.
